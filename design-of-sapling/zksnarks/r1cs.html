Notation

In order to explain how zk-SNARKs work, we'll borrow notation common in the Bulletproofs literature, as it may be familiar with readers.

  • We use uppercase letters like AAA, BBB, and CCC to describe elements of a group G\mathbb{G}G which is of prime order qqq. In practice these will be elements of either G1\mathbb{G}_1G​1​​ or G2\mathbb{G}_2G​2​​.
  • We use lowercase letters like aaa, bbb and ccc to describe scalars — elements of Fq\mathbb{F}_qF​q​​.
  • We use boldface to describe vectors: a\textbf{a}a is a vector of scalars and A\textbf{A}A is a vector of group elements. We use p(x)\textbf{p(x)}p(x) to describe vectors of polynomials with indeterminate xxx. We use xn\textbf{x}^nx​n​​ to describe vectors of scalars that are sequences of powers of some x∈Fqx \in \mathbb{F}_qx∈F​q​​.
  • We use the notation ⟨a,b⟩\langle \textbf{a}, \textbf{b} \rangle⟨a,b⟩ to describe the inner product of two vectors — in other words, the elements of each vector are multiplied pairwise, and the products are summed together. Note that ⟨a,b⟩\langle \textbf{a}, \textbf{b} \rangle⟨a,b⟩ produces a scalar and ⟨a,B⟩\langle \textbf{a}, \textbf{B} \rangle⟨a,B⟩ produces a group element.

R1CS

zk-SNARKs are zero-knowledge proofs which allow us to prove that we performed a computation f(p,w)f(p, w)f(p,w) over some witness www without revealing that witness. We express our computations in the form of arithmetic constraint systems.

Given an assignment z\textbf{z}z of variables in Fq\mathbb{F}_qF​q​​, a rank-1 constraint system is a system of quadratic constraints of the form a⋅b=ca \cdot b = ca⋅b=c, where a,b,ca, b, ca,b,c are linear combinations of our variable assignment. If we always set z0=1z_0 = 1z​0​​=1, then these constraint systems can express any bounded computation.

  • We can boolean constrain a variable ziz_iz​i​​ with the constraint (zi)⋅(zi)=(zi)(z_i) \cdot (z_i) = (z_i)(z​i​​)⋅(z​i​​)=(z​i​​), because squaring is the identity for only 000 and 111 in a field.
  • Given a boolean constrained variable ziz_iz​i​​, we can express ¬zi\neg z_i¬z​i​​ as z0−ziz_0 - z_iz​0​​−z​i​​.
  • Given two boolean constrained variables zi,zjz_i, z_jz​i​​,z​j​​, we can express zi∧zjz_i \land z_jz​i​​∧z​j​​ as zi⋅zjz_i \cdot z_jz​i​​⋅z​j​​.

As an example, consisider the constraint system:

  1. (z1)⋅(z1)=(z1)(z_1) \cdot (z_1) = (z_1)(z​1​​)⋅(z​1​​)=(z​1​​)
  2. (z2)⋅(z2)=(z2)(z_2) \cdot (z_2) = (z_2)(z​2​​)⋅(z​2​​)=(z​2​​)
  3. (2z1)⋅(z2)=(z1+z2−z3)(2z_1) \cdot (z_2) = (z_1 + z_2 - z_3)(2z​1​​)⋅(z​2​​)=(z​1​​+z​2​​−z​3​​)

The first two constraints are boolean constraints over z1z_1z​1​​ and z2z_2z​2​​, respectively. The third constraint is actually an XOR constraint: you'll see through substitution that z3z_3z​3​​ must equal z1⊕z2z_1 \oplus z_2z​1​​⊕z​2​​.

All of the terms of the constraint system are linear combinations of every variable; in other words, the above constraint system can be equivalently expressed with a zero coefficient for every omitted variable:

  • (−1z1+−0z2+−0z3)⋅(−1z1+−0z2+−0z3)=(−1z1+−0z2+−0z3)(\phantom{-}1z_1 + \phantom{-}0z_2 + \phantom{-}0z_3) \cdot (\phantom{-}1z_1 + \phantom{-}0z_2 + \phantom{-}0z_3) = (\phantom{-}1z_1 + \phantom{-}0z_2 + \phantom{-}0z_3)(−1z​1​​+−0z​2​​+−0z​3​​)⋅(−1z​1​​+−0z​2​​+−0z​3​​)=(−1z​1​​+−0z​2​​+−0z​3​​)
  • (−0z1+−1z2+−0z3)⋅(−0z1+−1z2+−0z3)=(−0z1+−1z2+−0z3)(\phantom{-}0z_1 + \phantom{-}1z_2 + \phantom{-}0z_3) \cdot (\phantom{-}0z_1 + \phantom{-}1z_2 + \phantom{-}0z_3) = (\phantom{-}0z_1 + \phantom{-}1z_2 + \phantom{-}0z_3)(−0z​1​​+−1z​2​​+−0z​3​​)⋅(−0z​1​​+−1z​2​​+−0z​3​​)=(−0z​1​​+−1z​2​​+−0z​3​​)
  • (−2z1+−0z2+−0z3)⋅(−0z1+−1z2+−0z3)=(−1z1+−1z2+−1z3)(\phantom{-}2z_1 + \phantom{-}0z_2 + \phantom{-}0z_3) \cdot (\phantom{-}0z_1 + \phantom{-}1z_2 + \phantom{-}0z_3) = (\phantom{-}1z_1 + \phantom{-}1z_2 + -1z_3)(−2z​1​​+−0z​2​​+−0z​3​​)⋅(−0z​1​​+−1z​2​​+−0z​3​​)=(−1z​1​​+−1z​2​​+−1z​3​​)

Let's begin to describe our constraint system using the inner product notation and coefficients represented by fixed vectors a,b,c\textbf{a}, \textbf{b}, \textbf{c}a,b,c:

  • ⟨a0,z⟩⋅⟨b0,z⟩=⟨c0,z⟩\langle \textbf{a}_0, \textbf{z} \rangle \cdot \langle \textbf{b}_0, \textbf{z} \rangle = \langle \textbf{c}_0, \textbf{z} \rangle⟨a​0​​,z⟩⋅⟨b​0​​,z⟩=⟨c​0​​,z⟩
  • ⟨a1,z⟩⋅⟨b1,z⟩=⟨c1,z⟩\langle \textbf{a}_1, \textbf{z} \rangle \cdot \langle \textbf{b}_1, \textbf{z} \rangle = \langle \textbf{c}_1, \textbf{z} \rangle⟨a​1​​,z⟩⋅⟨b​1​​,z⟩=⟨c​1​​,z⟩
  • ⟨a2,z⟩⋅⟨b2,z⟩=⟨c2,z⟩\langle \textbf{a}_2, \textbf{z} \rangle \cdot \langle \textbf{b}_2, \textbf{z} \rangle = \langle \textbf{c}_2, \textbf{z} \rangle⟨a​2​​,z⟩⋅⟨b​2​​,z⟩=⟨c​2​​,z⟩

More generally, our goal is to demonstrate that we know a satisfying assignment z=(1,p,w)\textbf{z} = (1, \textbf{p}, \textbf{w})z=(1,p,w) for which ⟨ai,z⟩⋅⟨bi,z⟩=⟨ci,z⟩\langle \textbf{a}_i, \textbf{z} \rangle \cdot \langle \textbf{b}_i, \textbf{z} \rangle = \langle \textbf{c}_i, \textbf{z} \rangle⟨a​i​​,z⟩⋅⟨b​i​​,z⟩=⟨c​i​​,z⟩ holds for all iii given fixed coefficients a,b,c\textbf{a}, \textbf{b}, \textbf{c}a,b,c. If we can do this without revealing w\textbf{w}w, and non-interactively with succinct proofs, we'll have a zk-SNARK.
